Tags para formatar o texto

Mark - Strong - b - i - em - u - abbr - ins - del - s - sup - sub

HTML semântico é uma abordagem de codificação que usa tags HTML para dar significado ao conteúdo de uma página web. O objetivo é que o conteúdo seja mais compreensível para os usuários, navegadores e motores de busca.

  • Mark: é usado para marcar ou destacar texto em um documento "devido à sua relevância em outro contexto" .
  • Strong: é usado para indicar que o texto é fundamental ou semanticamente importante deixando o texto em negrito.
  • b: deixa o texto em negrito sem importância.
  • i: deixa o texto em itálico sem ênfase (sem importância).
  • em: é usado para indicar que o texto deve ter uma ênfase(importância) extra que deve ser enfatizada e deixa o texto em itálico.
  • u: é usado para deixar o texto sublinhado
  • abbr: Para marcar alguma expressão como uma abreviação. Estamos estudando HTML !
  • ins: Para marcar o texto como inserido, ou seja foi inserido um novo text.
  • del: Para marcar o texto como excluído, ou seja texto deletado.
  • s: Para detalhar o texto.
  • sup - sub: Para definir o texto para cima ou para baixo.

    Para criar sobrescrito e Subescrito: Texto sobrescrito O texto sobrescrito é posicionado acima da linha normal de texto.

    Texto subescrito O texto subescrito é posicionado abaixo da linha normal de texto.
